# AI-Chatbot-Flask-Ollama
A simple local AI chatbot built with Flask, JavaScript and Ollama using the Llama 3.2 model.

#Features
- Chat interface with HTML, CSS and JavaScript
- Integration with Ollama and Llama 3.2
- Conversation history support
- Send messages by pressing Enter
- User and AI chat bubbles
- Automatic scrolling to the latest message

#Technologies Used
- Python
- Flask
- JavaScript
- HTML
- CSS
- Ollama
- Llama 3.2
